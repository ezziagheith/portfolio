My name is Ezzia and I am currently a student at General Assembly. I am enrolled in the 12 week software engineering immersive program. We learned many different frameworks and libraries like React, Bootstrap, Node, Express, Python, Django, and jQuery. Below are some project examples using different frameworks and libraries.

Revive and Thrive was made by one other partner and I.
We built this application in just one week using bootstrap,
node, express, mongoDB, mongoose, and javascript.
This was my first time using node and express, so it was
challenging given only five days to complete it, but it made
me a stronger backend developer. Revive and Thrive was made
to allow users to sign up, then get access to fitness events that were happeneing in San Francisco. Users are able to add the class to their list, then view their list on their account page.

Supreme Emperors was a project I did with three other partners. We used React and Node to build this website in one week. The instructions and prompt were given by our teachers. Learning react, then building a full-stack application right after was challenging as well, but definitly a great learning experience. Supreme Emperors allows users to register for an account, login, then be able to create posts on different cities around the world.
